## 5. Post-revision ED / 시나리오 분석 (PHASE 4)

| 시나리오 | 설명 | 평균 ΔSMP | 출력제한 |
|---------|------|----------|---------|
| Case_A_zero | 모든 블록 입찰가 0 | 0 원/MWh | 0 MWh |
| Case_B_floor | 모든 블록 최저가 | 0 원/MWh | 0 MWh |
| Case_C_mixed | 혼합 입찰 | 0 원/MWh | 0 MWh |
| Case_D_conservative | 보수적 입찰 | 0 원/MWh | 0 MWh |

**모든 시나리오에서 ΔSMP = 0, 출력제한 = 0.**

### 원인 분석

RE 입찰이 SMP에 영향을 미치지 못하는 이유:

1. **RE 비중 과소**: 2024년 RE 발전량(최대 ~7,500 MW)이 총 수요(60,000~97,000 MW)의 약 7~13%에 불과. 입찰 참여율 30% 적용 시 RE 입찰 물량은 ~2,250 MW로, 전체 급전 물량 대비 미미.
2. **잉여 용량 충분**: 총 설비용량 103,980 MW >> 최대 수요 97,115 MW. 열발전기만으로도 수요를 충분히 감당하므로, RE 입찰 블록이 thermal 급전 순서를 변경할 유인이 없음.
3. **Must-run 원전**: Nuclear 25기(~20,000~24,650 MW)가 must-run으로 항상 가동. 여기에 Coal 기저 투입까지 합하면, RE가 한계발전기를 변경할 여지가 제한적.

## 6. 민감도 분석 (PHASE 5)

### β (입찰가 하한 계수) 민감도
β = 1.5, 2.0, 2.5 모두 ΔSMP = 0. 입찰가 하한을 변경해도 RE 물량 자체가 한계발전기에 영향을 줄 만큼 크지 않음.

### ρ (참여율) 민감도
ρ = 0.1, 0.2, 0.3, 0.5 모두 ΔSMP = 0. 참여율 50%까지 높여도 RE 입찰 물량(~3,750 MW)이 급전 구조에 유의미한 변화를 만들지 못함.

### 몬테카를로 시뮬레이션
50회 시뮬레이션 결과: 모든 샘플에서 SMP 변화 없음. P5~P95 구간이 Pre SMP와 동일.

## 7. 핵심 발견 및 한계

### 발견 1: gencost 기반 MC와 실제 SMP의 괴리

gencost 비용함수(KPG193 스냅샷)에서 도출된 MC는 실제 2024년 SMP보다 구조적으로 낮음:
- 모형 평균 SMP: ~63,000~73,000 원/MWh
- 실제 평균 SMP: ~85,000~162,000 원/MWh

이는 gencost가 **특정 시점의 연료비를 반영한 고정 비용함수**이기 때문. 실제 시장에서는 월별 연료단가(원/Gcal) 변동이 SMP를 크게 좌우하나, 현 모형은 이를 반영하지 못함.

### 발견 2: 2024년 RE 비중으로는 입찰제 효과 미미

2024년 한국의 RE 설비용량(태양광 31,857 MW + 풍력 2,471 MW)에도 불구하고, 실제 발전량 기준 RE 비중이 낮아(이용률 문제) Post-revision ED에서 유의미한 SMP 변화가 관찰되지 않음.

### 발견 3: Calibration의 구조적 한계

Price Adder 보정이 MAE를 12.9% 개선하지만, gencost 기반 MC가 실제 연료비 수준을 반영하지 못하는 근본적 문제를 해소할 수 없음. Adder는 한계발전기(부분투입)에만 적용되므로, 전체 비용 수준의 괴리를 보정하기에 구조적 한계가 있음.

## 8. 개선 방향 제안

### 단기 (현 프레임워크 내)

1. **월별 연료비 반영**: gencost의 b 계수를 `(b_base × fuel_cost_actual / fuel_cost_base) × 1000`으로 시간가변 조정. fuel_costs.csv의 월별 열량단가를 활용하여 비용함수를 동적으로 스케일링.
2. **VOM 추가**: 한국전력거래소 정산 데이터에서 발전기별 VOM을 확보하여 MC에 가산.
3. **RE 시나리오 확대**: 2030년 RE 목표치(태양광 45 GW, 풍력 17 GW) 기준으로 RE 발전량을 스케일업하여 시뮬레이션. 현 2024년 수준에서는 RE 입찰 효과가 구조적으로 관측 불가.

### 중기

4. **Unit Commitment 추가**: 현 ED는 발전기 on/off 결정 없이 출력만 최적화. UC를 추가하면 기동/정지 비용이 반영되어 더 현실적인 SMP 도출 가능.
5. **송전제약 반영**: KPG193의 branch 데이터(363개 송전선)를 활용한 네트워크 제약 ED로 확장.
