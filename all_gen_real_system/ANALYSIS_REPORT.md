### 주요 코드 수정사항 (이전 버전 대비)

1. **원전 정비 반영**: 이름 기반 매핑 (고리4호기 -> Nuclear_003 등, 18건 전수 매칭)
2. **MC 계산**: 전력시장운영규칙 제2.4.2조에 따라 ED solve 후 실제 운전점에서 MC 재계산
   - 이전: `MC = (2a * (Pmin+Pmax)/2 + b) * 1000` (고정 중점, 휴리스틱)
   - 현재: `MC = (2a * P_actual + b) * 1000` (IP 수식 준수)
3. **heat_rate, vom 필드 완전 제거**: gencost 2차 비용함수에서 직접 MC 도출

### 한계연료 분석 (실제 운전점 MC 기반)

Pre-ED에서 한계연료를 **실제 운전점의 MC**로 식별한 결과, **모든 시간대에서 Coal이 한계연료**로 결정됨. 이는 전력시장운영규칙 제2.4.2조의 증분가격 수식 `IP = (2*QPC*DAOS + LPC) / TLF / 1000`을 적용했을 때:

- Coal 발전기: QPC(2차 계수)가 크고 (0.025~0.031), 실제 운전점에서 MC가 높아짐
- LNG 발전기: QPC가 상대적으로 작지만 (0.002~0.007), LPC(1차 계수)가 높음 (47~70)
- 실제 운전점에서 부분투입되는 Coal 발전기의 MC가 LNG보다 높게 산출

이는 Basic ED(중점 MC 기반)에서 Coal/LNG가 교차 결정되던 것과 대조적이며, **운전점 기반 MC가 한계연료 식별에 유의미한 차이를 만든다는 것을 보여줌**.

### 출력제한

12개 대표일 모두에서 RE 출력제한이 0. 2024년 RE 발전량(태양광 최대 6,255 MW, 풍력 최대 1,266 MW)이 총 수요(59,000~97,000 MW) 대비 매우 작아 초과공급이 발생하지 않음.

| Case_C_mixed | 혼합 입찰 | 0 원/MWh | 0 MWh |

**모든 시나리오에서 ΔSMP = 0, 출력제한 = 0.**

### 원인 분석

1. **RE 비중 과소**: 2024년 RE 발전량(최대 ~7,500 MW)이 총 수요의 약 7~13%에 불과. 입찰 참여율 30% 적용 시 RE 입찰 물량은 ~2,250 MW로 미미.
2. **잉여 용량 충분**: 총 설비용량 103,980 MW >> 최대 수요 97,115 MW. RE 입찰 블록이 thermal 급전 순서를 변경할 유인이 없음.
3. **Must-run 원전**: Nuclear 가용용량 19,450~21,400 MW가 must-run으로 항시 가동. 여기에 Coal 기저 투입까지 합하면 RE가 한계발전기를 변경할 여지가 제한적.

## 6. 민감도 분석 (PHASE 5)

### beta (입찰가 하한 계수) 민감도
beta = 1.5, 2.0, 2.5 모두 ΔSMP = 0. 입찰가 하한을 변경해도 RE 물량 자체가 한계발전기에 영향을 줄 만큼 크지 않음.

### rho (참여율) 민감도
rho = 0.1, 0.2, 0.3, 0.5 모두 ΔSMP = 0. 참여율 50%까지 높여도 RE 입찰 물량(~3,750 MW)이 급전 구조에 유의미한 변화를 만들지 못함.

### 몬테카를로 시뮬레이션
50회 시뮬레이션 결과: 모든 샘플에서 SMP 변화 없음. P5~P95 구간이 Pre SMP와 동일.

## 7. 핵심 발견

### 발견 1: 운전점 기반 MC가 한계연료 식별을 변화시킴

전력시장운영규칙의 증분가격 수식 `IP = (2*QPC*DAOS + LPC) / TLF / 1000`을 적용하면, 중점 MC 대비 한계연료 식별이 달라짐:

| 모형 | 한계연료 분포 (288시간) |
|------|----------------------|
| Basic ED (중점 MC) | Coal 168h, LNG 120h |
| Pre ED (실제 운전점 MC) | Coal 288h, LNG 0h |

Coal의 QPC(2차 계수)가 LNG 대비 5~7배 크기 때문에, 운전점에 따른 MC 변동 폭이 크고, 부분투입 시 MC가 LNG를 역전함.

### 발견 2: gencost 기반 MC와 실제 SMP의 구조적 괴리

| 구분 | 평균 (원/MWh) |
|------|-------------|
| 모형 SMP (Basic ED) | 62,653 |
| 모형 SMP (Pre ED, 보정 후) | 66,202 |
| 실제 SMP | 127,547 |

약 2배 차이. gencost가 KPG193 작성 시점의 연료비 스냅샷이기 때문.

### 발견 3: 2024년 RE 비중으로는 입찰제 효과 관측 불가

2024년 한국의 RE 설비용량(태양광 31,857 MW + 풍력 2,471 MW)에도 불구하고, 실제 발전량 기준 RE 비중이 낮아 Post-revision ED에서 유의미한 SMP 변화가 관찰되지 않음.

## 8. 개선 방향 제안

### 단기 (현 프레임워크 내)

1. **월별 연료비 스케일링**: gencost 계수를 `fuel_costs.csv`의 월별 열량단가로 동적 조정. 예: `a_scaled = a * (fuel_cost_2024 / fuel_cost_base)`, `b_scaled = b * (fuel_cost_2024 / fuel_cost_base)`.
2. **RE 시나리오 확대**: 2030년 RE 목표치(태양광 45 GW, 풍력 17 GW) 기준으로 RE 발전량을 스케일업. 현 2024년 수준에서는 RE 입찰 효과가 구조적으로 관측 불가.
3. **VOM 추가**: 한국전력거래소 정산 데이터에서 발전기별 변동운영비 확보하여 MC에 가산.

### 중기

4. **Unit Commitment 추가**: 현 ED는 발전기 on/off 결정 없이 출력만 최적화. UC를 추가하면 기동/정지 비용이 반영되어 더 현실적인 SMP 도출 가능.
5. **송전제약 반영**: KPG193의 branch 데이터(363개 송전선)를 활용한 네트워크 제약 ED로 확장.
